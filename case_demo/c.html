                              🌞 亮色主题 (Light Theme)
═══════════════════════════════════════════════════════════════════════════════

【背景色系】
  主背景      #F8F9FC    极淡灰蓝，柔和不刺眼
  卡片背景    #FFFFFF    纯白，干净通透
  次级背景    #F1F3F8    浅灰蓝，用于悬停、选中状态

【文字色系】
  主要文字    #2D3142    深灰蓝，非纯黑，降低视觉疲劳
  次要文字    #6B7280    中灰色，用于描述、标签
  辅助文字    #9CA3AF    浅灰色，用于时间戳、占位符

【边框色系】
  默认边框    #E5E7EB    浅灰边框
  悬停边框    #D1D5DB    略深的悬停状态

  次色        #7AB5B5    略亮的青色，用于悬停
  浅色背景    #E8F4F4    极浅青色，用于图标背景

【功能色系】
  收入绿      #6B9B7A    低饱和绿色，代表收入/正向
  收入浅      #E8F5EC    收入相关的浅色背景
  支出红      #C27B7B    低饱和红色，代表支出/负向
  支出浅      #FBEAEA    支出相关的浅色背景
  警告黄      #C9A856    低饱和黄色，代表预警/提醒
  警告浅      #FDF8E8    警告相关的浅色背景
  信息蓝      #7B9BC2    低饱和蓝色，代表信息/链接
  信息浅      #EAF0FB    信息相关的浅色背景

【阴影】

  悬停阴影    rgba(45, 49, 66, 0.12)

【圆角规范】

  大圆角      12px  (卡片)
  超大圆角    16px  (弹窗、面板)
  圆形        50%   (头像、状态点)

【间距规范】
  最小间距    4px
  小间距      8px
  中间距      16px
  大间距      24px
  超大间距    32px

【字体规范】
  字体族      'Inter', -apple-system, BlinkMacSystemFont, sans-serif
  大标题      24px / 600
  标题        18px / 600
  正文        14px / 400
  辅助文字    12px / 400
  最小文字    10px / 400

【动画规范】
  过渡时长    0.2s - 0.3s
  缓动函数    cubic-bezier(0.4, 0, 0.2, 1)
  悬停缩放    scale(1.02) - scale(1.1)

═══════════════════════════════════════════════════════════════════════════════
                              ✨ 使用建议
═══════════════════════════════════════════════════════════════════════════════

1. 收入相关数据使用绿色系，支出相关数据使用红色系
2. 品牌主色用于导航高亮、主要按钮、重要信息强调
3. 信息色用于链接、次要操作
4. 警告色用于预算超支提醒、异常数据标记
5. 图标建议使用 2px 描边线性风格，保持一致性
6. 卡片悬停时添加轻微上浮效果和阴影增强
7. 支持系统主题自动切换，提升用户体验
